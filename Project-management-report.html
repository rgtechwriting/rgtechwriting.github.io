Project Shore Line: Documentation Project

Overview

Format: Documentation plan
Length: 11 pages
Audience: Vice-president of a credit union's technology division
Tools used: Microsoft Word and Adobe Photoshop
Last updated: March 8, 2020

About

This document was created to fulfill the requirements for Simon Fraser University's TCOM320: Documentation Project Management course.

In TCOM320, students were required to create a documentation project plan for a hypothetical online banking software project. The course covered topics such as documentation content planning and resource management.

Project Shore Line: Documentation Project includes a documentation content plan, budget, and schedule.
